# Trình bày vai trò của [[Cuống (Stub)]] và [[Bộ lái (Driver)]] trong kiểm thử tích hợp.

| **Tiêu chí**    | **Stub**                                       | **Driver**                                 |
| --------------- | ---------------------------------------------- | ------------------------------------------ |
| **Định nghĩa**  | Chương trình giả lập thay thế module phụ thuộc | Chương trình đơn giản như hàm main         |
| **Khi sử dụng** | Top-down (từ trên xuống)                       | Bottom-up (từ dưới lên)                    |
| **Vai trò**     | Thay thế module bị gọi, cung cấp dữ liệu       | Khởi động, cung cấp input, kiểm tra output |
| **Mô phỏng**    | Hành vi của thành phần **bị gọi**              | Hành vi của thành phần **đang gọi**        |

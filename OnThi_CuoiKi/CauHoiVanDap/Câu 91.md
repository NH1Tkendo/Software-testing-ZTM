# Trong đoạn mã sau, hãy chỉ ra mã chết (dead code) và giải thích tại sao nó là một khiếm khuyết 

``` python
def check_value(x):
	if x > 0:
		return "Positve"
	elif x <= 0:
		return "Not Positive"
	print("This line will never be reached.")
```

**Dòng mã chết** trong đoạn code trên là `print("This line will never be reached.")`

Đây là khiếm khuyết Mã chết (không thể truy cập hoặc không bao giờ được thực thi) một dạng khiếm khuyết trong kiểm thử tĩnh. Mã này không có giá trị thực thi vì elif đã bao phủ toàn bộ nhánh có thể chạy được, vi phạm tiêu chuẩn lập trình, đoạn mã này có thể gây hiểu nhầm vào khó bảo trì, gây lãng phí tài nguyên

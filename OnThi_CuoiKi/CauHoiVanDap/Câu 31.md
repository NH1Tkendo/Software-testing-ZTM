# Cho đoạn mã sau: if (x > 10 and y < 100); do_something(). Hãy thiết kế một bộ ca kiểm thử tối thiểu để đạt được Bao phủ điều kiện (Conditional Coverage)

Để **đảm bảo tất cả điều kiện con được đánh giá thực sự**, bộ test case tối thiểu nên là:

- **TC1: x = 11, y = 99** → `x > 10` = T (được đánh giá), `y < 100` = T (được đánh giá)
    
- **TC2: x = 9, y = 99** → `x > 10` = F (được đánh giá), `y < 100` = T (được đánh giá)
    
- **TC3: x = 11, y = 101** → `x > 10` = T (được đánh giá), `y < 100` = F (được đánh giá)
